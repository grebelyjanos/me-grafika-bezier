# me-grafika-bezier
 ## 2D bezier görbe rajzolása

 A program célja 2D bezier görbe kirajzolása.
 Maximum pontok száma PMSIZE makróban definiálható.
 
 A program alapvető interrakcióra képes a lentebb írtak alapján
 ```
 q - Kilépés
 e - Random számú kontrolpont generálása
 r - Random kontrolpontok generálása
 a - Kontrolpontok számának növelése
 s - Kontrolpontok számának csökkentése
 ```
